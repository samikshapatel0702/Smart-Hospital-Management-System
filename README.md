# 🏥 Smart Hospital Management System

A console-based Hospital Management System developed in Python using Object-Oriented Programming (OOP) concepts. The system helps manage patients, doctors, nurses, appointments, pharmacy records, medical records, laboratory reports, billing, and hospital statistics through a menu-driven interface.

## 🚀 Features

* Patient Management

  * Register new patients
  * Update patient details
  * Search patients
  * View medical history

* Doctor Management

  * Add and manage doctors
  * View doctor information
  * Manage doctor schedules

* Nurse Management

  * Add and manage nurses
  * Shift management

* Appointment Management

  * Book appointments
  * View appointment records

* Pharmacy Management

  * Manage medicine inventory
  * Track medicine stock

* Medical Records

  * Store patient medical history
  * Maintain treatment records

* Laboratory Reports

  * Generate and manage lab reports

* Billing Management

  * Generate patient bills
  * Calculate hospital charges

* Data Persistence

  * Save and load data using JSON files

* Reports & Statistics

  * Generate CSV reports
  * View hospital statistics
